# Traffic Light Controller API
Spring Boot REST API to control traffic lights at intersections.

By Praveen Singh !!!

Features
- Manage traffic light states
- Prevent conflicting green lights
- Pause / Resume traffic control
- Automatic scheduler
- Traffic history tracking
- Concurrent safe design
- Future support for multiple intersections

Technologies

- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven
